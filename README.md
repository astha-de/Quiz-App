# Quiz App 

A simple and interactive Quiz App made using HTML, CSS, and JavaScript.

## Features 

- Multiple choice questions
- Next button navigation
- Correct and incorrect answer highlight
- Dynamic questions using JavaScript
- Simple and responsive UI

## Technologies Used 

- HTML
- CSS
- JavaScript

## How to Run 

1. Download or clone the project
2. Open the project folder
3. Run `index.html` in your browser

## Project Structure 

quiz-app/
│
├── index.html

└── script.js
